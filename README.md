1. What is JSX, and why is it used?
 -> JSX stands for JavaScript XML.
It lets us write HTML inside JavaScript.

2. What is the difference between State and Props?
 -> State Used to store data inside a component
 -> Props (Properties) Used to pass data from parent component to child component

3. What is the useState hook and how does it work?
 -> useState is a React Hook used to store and update state in functional components.

4. How can you share state between components in React?
 -> we have to share state by lifting the state up to the parent component and passing it to child components using props.

5. How is event handling done in React?
 -> React handles events using camelCase event names and functions.
